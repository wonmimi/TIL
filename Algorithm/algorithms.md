### 정렬
- 퀵정렬 O(n*log(n)) , 선택정렬 O(n2)

### 탐색
- 이진탐색 O(log n) , 단순탐색 O(n)

### 탐색 - 그래프
데이터 탐색에 해시 테이블 자료구조 사용

- BFS (너비 우선 탐색) (O(V+ex)) 최단경로 찾기 (균일그래프)
  : 탐색시 큐를 이용
- DFS
- 다익스트라 : 최단 가중치합 최소 찾기 (가중그래프) 
  : 가중치는 양수.
- 탐욕 알고리즘 (greedy)

### 자료구조
- 해시 테이블 : key,value 
  ( 해시 맵, 딕셔너리 , 연관배열 )
- 스택 : LIFO , 탐색엔 쓰지않음 ex) 호출 스택
- 큐 : FIFO
